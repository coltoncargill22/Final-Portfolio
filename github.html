What I've Learned

Github

In this course we were introduced to Github. Github is a tool used to help devlopers work on, edit, and publish their work. We used it a lot in this class especially on our social media page project where we all used Github to work on our own sections of the site and we were able to upload our work at the end of the project and put all our work together to complete the site.

Why it's important

Github is a very useful tool to anyone who thinks they would like to start learning web devolpoment. Github can help you keep track of your work and when you are finished, you can publish it. But github is not only a something uses in schools to help teach students how devolopment works. Github is used in most fields of work today, if you want to pursue a job in devlopment learning Github early on is one of the best things you can do to help your future carear.
